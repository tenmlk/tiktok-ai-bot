# 🤖 بوت تيك توك الذكي

بوت تيك توك يتفاعل مع الفيديوهات بالذكاء الاصطناعي، يمنشن صاحب الفيديو، يعطي رأيه ومعلومات، **ويرفض تماماً** التفاعل مع المحتوى الحساس.

## ✨ المميزات

- 🎯 **منشن تلقائي** - يبدأ تعليقه بـ @username دائماً
- 🧠 **ذكاء اصطناعي** - يولد ردود ذكية ومناسبة لكل فيديو
- 🛡️ **فلترة محتوى** - يرفض التفاعل مع التحريض، المخدرات، والمواضيع الطبية
- 🎭 **شخصيات متعددة** - ودود، فكاهي، ناقد بنّاء، مثقف
- 📊 **إحصائيات** - يتتبع كل التفاعلات
- ⏱️ **حماية من الحظر** - تأخير عشوائي بين التفاعلات

## 🚀 التشغيل السريع

### 1. تثبيت المكتبات

```bash
cd tiktok-bot
npm install
```

### 2. إعداد الإعدادات

```bash
cp .env.example .env
# عدّل ملف .env وحط معلوماتك
```

### 3. تشغيل البوت

```bash
# تشغيل عادي
npm start

# وضع المحاكاة (للاختبار بدون نشر حقيقي)
node src/index.js --mode=simulate

# تحديد هاشتاق معين
node src/index.js --hashtag=طبخ

# تغيير الشخصية
node src/index.js --personality=funny

# تفعيل الرد على التعليقات
node src/index.js --interact=true
```

## ⚙️ الإعدادات

| المتغير | الوصف | القيمة الافتراضية |
|---------|-------|------------------|
| `TIKTOK_USERNAME` | اسم حساب البوت | - |
| `TIKTOK_PASSWORD` | كلمة مرور البوت | - |
| `MAX_VIDEOS_PER_RUN` | عدد الفيديوهات لكل تشغيل | 10 |
| `DELAY_BETWEEN_ACTIONS` | التأخير بين التفاعلات (ثانية) | 30 |
| `BOT_LANGUAGE` | لغة الردود | ar |
| `BOT_PERSONALITY` | شخصية البوت | friendly |
| `TARGET_HASHTAGS` | هاشتاقات البحث | funny,comedy... |
| `TARGET_USERS` | مستخدمين محددين | - |
| `SAFE_MODE` | تفعيل الفلترة | true |

## 🎭 الشخصيات المتاحة

| الشخصية | الوصف |
|---------|-------|
| `friendly` | ودود ومشجع (الافتراضي) |
| `funny` | مرح وخفيف |
| `critic` | صريح ومحترم |
| `informative` | مثقف ومفيد |

## 🛡️ فلترة المحتوى

البوت يرفض التفاعل مع:

- ❌ **التحريض والكراهية** - أي دعوة للعنف أو الكراهية
- ❌ **المخدرات والمسكرات** - أي ذكر للمخدرات أو المسكرات
- ❌ **المواضيع الطبية** - نصائح طبية أو تشخيصات
- ❌ **العنف والتهديد** - تهديدات أو تحريض على الأذى

عند اكتشاف محتوى حساس، البوت يعطي رد مهذب يرفض الموضوع.

## ⚠️ تنبيهات مهمة

1. **لا تستخدم البوت للسبام** - التفاعل المفرط قد يسبب حظر الحساب
2. **احترم شروط خدمة تيك توك** - البوت مخصص للتفاعل الإيجابي فقط
3. **الفلترة ليست مثالية 100%** - راقب ردود البوت بشكل دوري
4. **تأخر بين التفاعلات** - لا تقلل التأخير عن 30 ثانية
5. **راجع التعليقات** - تأكد إن ردود البوت مناسبة

## 📌 ملاحظة تقنية

بوت تيك توك يستخدم مكتبة `tiktok-scraper` غير الرسمية لأن تيك توك لا يوفر API رسمي للبوتات. في حال تغيرت API تيك توك، قد تحتاج لتحديث المكتبة أو استخدام طرق بديلة.

البوت يعمل في **وضع المحاكاة** افتراضياً لحماية حسابك. لتشغيله بشكل حقيقي، سجل دخولك في ملف `.env`.
